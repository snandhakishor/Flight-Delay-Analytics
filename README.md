## Flight Delay Analytics

This project analyzes flight delay patterns across airlines, airports, and time periods.

 **Detailed analysis and insights** are documented in:
- `docs/Project Explanations.pdf`

### Summary
- Focuses on delay frequency vs delay duration
- Uses interactive Power BI visuals with slicer-driven exploration
- Highlights cases where fewer delays still result in high operational impact

### Notes
This repository contains the Power BI report and supporting assets. The PDF provides the full analytical narrative.
